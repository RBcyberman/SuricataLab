<h1> Suricata Home Lab </h1>

<h2>Description</h2>
Create Suricata IDS/IPS rules (signatures) to detect below suspicious activities (or other IDS detection rules of personal choice). Add these rules to the Suricata custom rules file. Create a test scenario for each suspicious activity using a remote machine (it can be another VM in your home lab). Prepare a report of the lab exercises and include the rules, an explanation about each rule, Suricata configuration changes to enable these rules, and detection results by investigating the log files (eve.json, fast.log locally)
<br />


<h2>Languages and Utilities Used</h2>

- <b>Suricata</b> 
- <b>Custom Snort rules</b>
- <b>Nano editor</b>

<h2>Environments Used </h2>

- <b>Kali Linux</b> 
- <b>VM Prostation</b>
